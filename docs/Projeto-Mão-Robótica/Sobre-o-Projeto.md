**Objetivos:**
Nossa ideia inicial é criar uma mão robótica capaz de medir e coletar dados vitais do usuário. Ela contará com sensores de pressão arterial, oxigenação do sangue, temperatura e frequência cardíaca. A mão será integrada a um aplicativo de apoio para auxiliar pessoas, especialmente idosos ou indivíduos com limitações, permitindo a realização de medições básicas de saúde.
Ela é um dispositivo de upgrade para robôs humanoides, com futuras previsões para ser uma prótese.
