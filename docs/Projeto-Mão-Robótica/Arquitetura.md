# Tecnologia


**Frontend:** Ionic/angular/CSS

**Backend:** TypeScript/JavaScript

**Sistema embarcados:** 
Placa ESP-32(DEVkIT)
*   16 Motor-Servo MG90S
*   2 Motor Servo MG996R
*   2 Módulo PCA9685
*   Fonte Chaveada 5V, 20A
*   LED’s
*   Sensores:
> Módulo Sensor De Frequência Cardíaca E Oxigênio Max30102
> Sensor de Temperatura IR MLX90614

**Matérias:**

*   Material: Filamento PLA
*   Rolamentos de metal
*   Molas de tração
*   Tubo PTFE
*   Linha de Pesca 0,30 mm
*   Linha de Pedreiro
*   Parafusos e pregos
*   Pino de Metal

**Impressão 3D**

A estrutura da mão robótica é feita totalmente em pla, usando uma impressora 3d, arquivo feito em modelagem .stl.
