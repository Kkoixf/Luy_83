
# Luy-83-CodeQueens

_Version English_

** This project presents the Luy-83, a robotic hand inspired by human mobility, designed for manipulation testing and biometric control. With multiple servo motors, the hand enables detailed movements of the fingers and palm, while embedded sensors capture tactile and pressure information. It is ideal for applications in assistive robotics. **

_Version Português_

** Este projeto apresenta a Luy-83, uma mão robótica inspirada na mobilidade humana, projetada para testes de manipulação e controle biométrico. Com múltiplos servos motores, a mão permite movimentos detalhados dos dedos e da palma, enquanto sensores embutidos capturam informações táteis e de pressão. Ideal para aplicações em robótica assistiva. **
